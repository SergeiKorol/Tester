# Tester
Приложенька для оффлайн тестов

Работает и для ПК и для Андроид, другие ОС не проверял.

Открываем файл index.html в браузере и пользуемся. 

Чтоб создать свой тест добавляем файл с тестами по аналогии с oop.js manual.js и добавляем строку о файле в manifest.js

Чтоб статистика считалась правильно нумерация вопросов не должна пересекаться.

Чтоб вернуться на главную страницу обновляем страницу
